## 数据操作接口使用方式
当数据从微信服务器传送过来，服务器自身访问接口对数据进行存储（用户发送的明文消息等）
当发生用户行为，根据不同的用户行为调用不同的接口对数据进行存储。
后台系统可以调用数据接口访问用户交互数据
后台系统在对用户做出数据响应的行为后，追加用户交互数据
后台系统不存在主观意识上的数据修改行为
服务器调用的接口与后台系统调用的接口不能使用相同的接口，以免造成混淆
